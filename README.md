# 🕹️ Meza Digital Command Center (Sentry)

Unified, zero-cost management hub for Hjalmar Meza's digital ecosystem.

## 🚀 Key Features
- **Project Monitoring:** Real-time stats for 20+ GitHub repositories.
- **Strategic Research:** Advanced lead and job search operators.
- **Brand Management:** Personal URL shortener and QR generator.
- **Infrastructure Control:** Remote management of web services.

## 🛠️ Stack
- **Engine:** Node.js (Serverless)
- **Deployment:** Vercel (Edge)
- **Integration:** Telegram Bot API & GitHub API

## 🔒 Security
Access restricted to authorized User ID only.
